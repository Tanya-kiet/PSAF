# PSAF — Prompt Stability Analysis Framework

## Quick Start

```bash
pip install -r requirements.txt
export GROQ_API_KEY="gsk_..."          # Set your key here (never in code)
streamlit run streamlit_app.py
```

---

## 🔐 API Key Security Rules

### Where to store your key

**Option A — Shell export (recommended for development):**
```bash
export GROQ_API_KEY="gsk_yourkey"
# Add this line to ~/.bashrc or ~/.zshrc to persist it
```

**Option B — `.env` file (local only, NEVER commit):**
```
GROQ_API_KEY=gsk_yourkey
```
Then load with `python-dotenv` or `direnv`.

### Files you must NEVER share or upload

| File / Folder | Why |
|---|---|
| `.env` | Contains your API key |
| `.psaf_cache/` | Contains LLM response data |
| `venv/` | Not needed by others |
| `__pycache__/` | Auto-generated bytecode |
| `.DS_Store` | macOS metadata |

### `.gitignore` (add these lines)
```
.env
.psaf_cache/
venv/
__pycache__/
*.pyc
.DS_Store
```

## Architecture

```
streamlit_app.py   ← UI only, no business logic
backend.py         ← All Groq calls, PSI computation, disk cache
config.py          ← Settings, categories, API key read
requirements.txt   ← Dependencies
```

### How API calls are minimized

1. **One "Run" = one batch**: clicking Run generates all variations + responses in one go.
2. **Disk cache**: results saved to `.psaf_cache/<hash>.json`. Same question + same n_variations = instant result, zero Groq calls.
3. **Session cache**: results stored in `st.session_state`. Tab switching never re-calls Groq.
4. **Model singleton**: SentenceTransformer loaded once per process.

### PSI Score

```
PSI = 100 × (0.50 × Semantic + 0.30 × Keyword + 0.20 × Length)
```

- **75–100**: Highly stable
- **50–74**: Moderately stable  
- **0–49**: Unstable
